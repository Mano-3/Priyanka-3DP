Keka HR Analytics Dashboard

Dashboard Design HRMS Keka 2023

Overview

Create. Design. Develop.

KEKA HR with a customisable dashboard is a powerful tool that allows HR professionals, managers, and even employees to gain quick, tailored insights into various HR data points.

Insights & Add Widget

Designed and deployed a highly interactive HRMS dashboard. Created a user-centric HRMS that fosters real-time collaboration through integrated chat and commenting functionalities, alongside customisable widgets for personalised data for enhanced collaboration and informed decision-making.

Internal Team Sharing

Threaded commentary system & conversational workflows: Allowing users to initiate and build threaded conversations directly on specific metrics for granular analysis and accelerated problem-solving.

Shared Dashboard

Granular Role-Based Access Control

Control over all dashboards, sharing settings and user permissions.

Data Security & Confidentiality

Sharing publicly within Keka, sensitive data will be masked to maintain privacy compliance.
